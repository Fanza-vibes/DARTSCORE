# DARTS SCORE

App web per il calcolo dei punteggi nelle partite di freccette.

## Descrizione

Darts Score è un'applicazione single-page pensata per calcolare in modo rapido e preciso i punteggi di una partita di freccette, gestendo automaticamente set, leg, squadre e giocatori.

### Funzionalità principali

- Gestione completa di **squadre e giocatori**
- Calcolo automatico di **set e leg** secondo il regolamento ufficiale
- Validazione in tempo reale degli input
- Riepilogo dettagliato a fine partita
- Interfaccia mobile-first, ottimizzata per l'uso a bordo campo
- Funziona interamente offline — nessun server richiesto
- Nessun dato inviato a terzi — tutto resta nel browser

## Come scaricarlo

### Metodo rapido — scarica solo il file dell'app

1. Vai alla pagina del progetto su GitHub: [github.com/Fanza-vibes/DARTSCORE](https://github.com/Fanza-vibes/DARTSCORE)
2. Cerca il file **`Darts_Score_1_9.html`** nella lista dei file e cliccaci sopra
3. Clicca il pulsante **"⬇ Download raw file"** (icona con freccia verso il basso, in alto a destra nella pagina del file)
4. Il file verrà salvato nella cartella *Download* del tuo dispositivo

### Metodo alternativo — scarica tutto il progetto

1. Dalla [pagina principale del progetto](https://github.com/Fanza-vibes/DARTSCORE), clicca il pulsante verde **"<> Code"**
2. Seleziona **"Download ZIP"**
3. Apri il file ZIP scaricato ed estrai il contenuto in una cartella a tua scelta
4. All'interno troverai il file `Darts_Score_1_9.html` insieme a tutti gli altri file del progetto

> **Nota:** non serve installare nulla. Il file `.html` si apre direttamente con qualsiasi browser (Chrome, Safari, Firefox, Edge).

## Come usarlo

1. Apri il file `Darts_Score_1_9.html` con un doppio clic (o tieni premuto su telefono e scegli "Apri con" → il tuo browser)
2. Configura squadre e giocatori
3. Inserisci i punteggi turno per turno
4. Consulta il riepilogo a fine partita

> Nessuna installazione necessaria. Basta un browser.

L'applicazione non raccoglie, trasmette o condivide alcun dato personale. Tutti i dati della partita vengono gestiti localmente nel browser e non vengono mai inviati a server esterni. Per maggiori dettagli, consulta l'informativa all'interno dell'app.
